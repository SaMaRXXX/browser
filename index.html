ZenithWeb

An Interactive Showcase of a Secure Web Browser Project

Project Introduction

This showcase presents ZenithWeb, a functional and secure desktop web browser built from the ground up using Python and the powerful PyQt5 framework. The project's core philosophy is to prioritize user security by strictly enforcing HTTPS connections for all web traffic, ensuring a safer Browse experience.

Project Goals

  • Demonstrate desktop application development.
  • Explore common features of modern browsers.
  • Highlight the importance of secure communication.

Why ZenithWeb?

This project serves as a practical example of integrating complex web technologies within a native desktop application, providing a tangible platform to learn about UI/UX design, application logic, and web security principles.

Future Enhancements Roadmap

ZenithWeb is a foundational project with significant potential for growth. The following roadmap outlines planned features that will further enhance its functionality, security, and user experience.

Ad Blocker & Content Filtering

Integrate a content filtering system to block advertisements and trackers for a cleaner, faster, and more private Browse experience.

Incognito / Private Browse Mode

Implement a private Browse mode that does not save history, cookies, or site data, ensuring user privacy for sensitive sessions.

Robust Download Manager

Expand the current download functionality to include progress details, pause/resume capabilities, and custom save locations.

Cookie Management

Provide a user interface to view, search, and delete cookies on a per-site or global basis, giving users more control over their data.

Theming and Customization

Allow users to personalize the browser's appearance with different color themes and layout options.
